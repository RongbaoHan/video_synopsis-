这个视频浓缩主要分为以下的几个步骤

1、多目标检测+跟踪，确定序号和轨迹
形成每个跟踪对象轨迹的索引
2、确定视频编辑的背景
3、检查跟踪轨迹的重合，避免多个对象在同一时刻出现在同一位置，造成重叠
4、形成一个新的索引，并依据这个索引生成视频浓缩

# 一些必要的准备工作
